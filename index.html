Step-by-Step Setup

Follow these steps in order. Fire it up and let's get you protected.

01

Get Firefox

Start with a browser that actually respects you. Firefox is fast, secure.

  1. Grab the installer from mozilla.org/firefox
  2. Run it and follow the quick setup wizard
  3. Launch Firefox—this is where your new web starts
02

Add uBlock Origin

Shields up. This is the only extension you truly *need* for a clean web.

  1. Open uBlock Origin on Add-ons Store
  2. Click "Add to Firefox"
  3. Look for the icon in your toolbar—it's already working
03

Enable Violentmonkey

This is your foundation for running custom scripts that "fix" broken site behaviors.

  1. Go to the Violentmonkey page
  2. Click "Add to Firefox" and confirm
04

Deploy the Link Bypasser

No more waiting for timers. This script handles the "Wait 10s" garbage for you.

  1. Click here to install directly via Violentmonkey:
    [One-Click To Install Bypass Script]
  2. Violentmonkey will pop up—just click "Confirm Installation"
  3. You're all set. No more timers.

Pro Tip: Keep an eye on Amm0ni4's page for updates.

05

Configure & Enjoy

Fine-tune your setup. Adjust settings, add custom filters, and explore advanced features once you're comfortable.

  • Enable extra filter lists in uBlock for maximum protection
  • Whitelist sites you actually want to support
  • Check Greasyfork for more scripts
  • Enjoy a web that actually works for you!
