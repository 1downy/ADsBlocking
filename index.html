Setup Guide

Follow these steps in order to set up your complete ad-free browsing experience

01

Install Firefox Browser

Start by downloading Firefox, a browser built for privacy and customization. It's fast, secure, and doesn't track your every move.

  1. Visit mozilla.org/firefox
  2. Download the installer for your operating system
  3. Run the installer and follow the setup wizard
  4. Launch Firefox and make it your default browser (optional)
02

Install uBlock Origin

Add the most effective ad blocker available. uBlock Origin blocks ads, trackers, and malware while using minimal system resources.

  1. Open Firefox and navigate to the uBlock Origin add-on page
  2. Click "Add to Firefox"
  3. Confirm the installation when prompted
  4. The uBlock Origin icon will appear in your toolbar
  5. Click the icon to access settings and customize your blocking preferences
03

Install Violentmonkey

Enable userscripts to customize websites and add extra functionality. This is required for the link bypasser scripts to work.

  1. Visit the Violentmonkey add-on page
  2. Click "Add to Firefox"
  3. Confirm the installation
  4. The Violentmonkey icon will appear in your toolbar
04

Install Link Bypasser Scripts

Install Amm0ni4's link bypasser scripts to automatically skip annoying short-link pages. Important: Violentmonkey must be installed first!

  1. Make sure you've completed Step 3 (Violentmonkey installation)
  2. Visit Amm0ni4's GitHub page
  3. Browse their userscript repositories for link bypasser scripts
  4. Click on a script you want to install
  5. Click the "Install" or "Raw" button - Violentmonkey will detect the script
  6. Confirm the installation in the Violentmonkey popup
  7. The script will now automatically bypass supported short-link sites

Warning: If you try to install scripts without Violentmonkey, they won't work!

05

Configure & Enjoy

Fine-tune your setup for the best experience. Adjust settings, add custom filters, and explore advanced features as you get comfortable.

  • In uBlock Origin, enable additional filter lists for extra protection
  • Whitelist trusted sites to support creators you love
  • Install additional userscripts from Greasyfork that match your needs
  • Regularly update your extensions for the latest features and security patches
  • Enjoy a cleaner, faster, and more private browsing experience!
